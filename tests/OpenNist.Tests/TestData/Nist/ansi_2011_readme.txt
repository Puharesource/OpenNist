The contents of this ZIP file include ANSI/NIST-ITL 1-2011 sample data
files and some of the tools used to create and test them.

The sample data files are in 'samples/traditional' and
'samples/xml'.  The samples included are described briefly in
'samples/samples.doc'.

The source images, txt2an2k specification files, Perl scripts,
and Unix-style Makefile are also within 'samples' and its
sub-folders.  Common Unix utilities that are required to generate
the samples, like 'make', and 'xmllint' are not included.

A copy of the NBIS distribution is included in 'Rel_4.0.1'.
It is slightly modified in order to handle some of the new
features of ANSI/NIST-ITL 1-2011.

Two versions of schemas for the XML files, used to validate the
XML files, are included in 'ansi-nist-itl-2011.iepd' and
'fbi_ebts'.

This data was compiled by Joe Konczal on July 16, 2012.
