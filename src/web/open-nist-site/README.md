# OpenNist Web App

Browser workbench for viewing, inspecting, and converting WSQ files.

Toolchain:

- `Vite 8`, which uses `Rolldown`
- `@vitejs/plugin-react`, which uses `Oxc`
- `Oxlint` for linting
- `Bun` for package management and scripts

Useful commands:

```bash
bun run dev
bun run lint
bun run build
```
